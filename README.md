# JobHandaAI - AI-Powered Interview Practice Platform

JobHandaAI is a comprehensive web application that helps job seekers practice and prepare for interviews using AI-powered coaching. The platform supports HR interviews, technical interviews, and LeetCode-style coding challenges.

## Features

### For Users
- **AI-Powered Interview Practice**: Practice with realistic questions generated by AI
- **Multiple Interview Types**:
  - HR Behavioral Interviews
  - Technical Interviews
  - LeetCode Coding Challenges
- **Instant Feedback**: Get detailed AI feedback on your answers
- **Progress Tracking**: Monitor your improvement over time
- **Session History**: Review past sessions and learn from feedback
- **Score Analytics**: Track your scores and performance metrics

### For Admins
- **User Management**: View and manage all registered users
- **Session Monitoring**: Track all interview sessions across the platform
- **System Analytics**: View comprehensive statistics and metrics
- **Settings Management**: Configure system settings

## Technology Stack

- **Backend**: PHP 7.4+
- **Database**: MySQL 5.7+
- **Frontend**: HTML5, CSS3, JavaScript
- **AI Integration**: Hugging Face API (Llama 3.1 8B Instruct)
- **Design**: Custom CSS with modern, professional styling

## Installation

### Prerequisites

1. **Web Server**: Apache or Nginx
2. **PHP**: Version 7.4 or higher
3. **MySQL**: Version 5.7 or higher
4. **Hugging Face Account**: For API access

### Step-by-Step Setup

#### 1. Clone or Download Files

Place all files in your web server's document root (e.g., `htdocs` for XAMPP, `www` for WAMP).

#### 2. Database Setup

1. Create a new MySQL database:
```sql
CREATE DATABASE jobhandaai;
```

2. Import the database schema:
   - Open phpMyAdmin or your MySQL client
   - Select the `jobhandaai` database
   - Import the `database.sql` file
   
   OR run it via command line:
```bash
mysql -u root -p jobhandaai < database.sql
```

#### 3. Configure Database Connection

Edit `/includes/config.php` and update the database credentials:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_mysql_username');
define('DB_PASS', 'your_mysql_password');
define('DB_NAME', 'jobhandaai');
```

#### 4. Configure Hugging Face API

1. Get your Hugging Face API token:
   - Go to https://huggingface.co/settings/tokens
   - Create a new token or copy an existing one

2. Update the API token in `/includes/config.php`:

```php
define('HF_TOKEN', 'your_hugging_face_token_here');
```

#### 5. Set File Permissions

Ensure the web server has read/write permissions:

```bash
chmod -R 755 /path/to/JobHandaAI
chown -R www-data:www-data /path/to/JobHandaAI
```

#### 6. Access the Application

Open your browser and navigate to:
```
http://localhost/JobHandaAI/
```

## Usage Guide

### For Users

1. **Register**: Create a new account at `/register.php`
2. **Login**: Access your dashboard
3. **Start Practice**: Choose interview type (HR, Technical, or LeetCode)
4. **Answer Questions**: Type your responses to AI-generated questions
5. **Get Feedback**: Receive instant AI evaluation and scoring
6. **Review Progress**: Check your dashboard for statistics and history

### For Admins

1. **Change User Role**: To make a user an admin, update the database:
```sql
UPDATE users SET role = 'admin' WHERE username = 'username_here';
```

2. **Access Admin Panel**: Login with admin credentials
3. **Monitor Users**: View all registered users and their progress
4. **Track Sessions**: Monitor all interview sessions
5. **View Analytics**: Check system-wide statistics

## Database Schema

4. **user_progress**: User statistics and progress tracking

## API Integration

The application uses the Hugging Face API for AI-powered features:

- **Model**: meta-llama/Llama-3.1-8B-Instruct:novita
- **Endpoint**: https://router.huggingface.co/v1/chat/completions

### API Usage

The system makes API calls to:
1. Generate interview questions based on type
2. Evaluate user answers and provide feedback
3. Calculate scores and correctness

## Security Considerations

1. **Password Hashing**: Uses PHP's `password_hash()` with bcrypt
2. **SQL Injection Protection**: Prepared statements for all queries
3. **Input Sanitization**: All user inputs are sanitized
4. **Session Management**: Secure session handling
5. **Access Control**: Role-based authentication

### Security Recommendations

- Change default admin password immediately
- Keep your Hugging Face API token secret
- Use HTTPS in production
- Regularly backup your database
- Update PHP and MySQL to latest versions
- Set proper file permissions

## Troubleshooting

### Common Issues

**1. Database Connection Failed**
- Check MySQL is running
- Verify database credentials in `config.php`
- Ensure database exists

**2. AI Questions Not Generating**
- Verify Hugging Face API token is correct
- Check API token has proper permissions
- Ensure server can make external API calls

**3. Sessions Not Saving**
- Check database permissions
- Verify session table exists
- Check PHP session configuration

**4. CSS/JS Not Loading**
- Verify file paths are correct
- Check web server configuration
- Clear browser cache

## Customization

```css

    --accent: #e8a838;

    /* ... other colors */
```

### Adding Interview Types

1. Update the `interview_sessions` table enum
2. Add new prompts in `/user/api/generate_question.php`
3. Add evaluation logic in `/user/api/submit_answer.php`
4. Update UI to show new type
